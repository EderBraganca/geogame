# GeoGame
GeoGame is an interactive geography game that challenges players to test their knowledge about the world. The objective of the game is to click as close as possible to a specific location on the map, earning points based on the accuracy of the click. The game is designed to be fun and educational, helping players learn about different countries, cities, and points of interest around the globe.

## Features
- Intuitive and responsive interface
- Scoring system based on click accuracy
- Visual feedback for each attempt
- Multiple rounds to challenge players
- Modern and attractive design

## Technologies Used
- React for building the user interface
- CSS for styling
- React Router for page navigation
- React Icons for visual icons
- React Hash Link for smooth in-page navigation
- React Redux for state management

## How to Install and Run the Project
1. Clone the repository;
2. Navigate to the project directory;
3. Install dependencies using `npm install`;
4. Start the development server with `npm run dev`;
