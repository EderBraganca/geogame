# GeoGame

GeoGame é um jogo de geografia interativo que desafia os jogadores a testarem seus conhecimentos sobre o mundo. O objetivo do jogo é clicar o mais próximo possível de um local específico no mapa, acumulando pontos com base na precisão do clique. O jogo é projetado para ser divertido e educativo, ajudando os jogadores a aprenderem sobre diferentes países, cidades e pontos de interesse ao redor do globo.

## Funcionalidades
- Interface intuitiva e responsiva
- Sistema de pontuação baseado na precisão do clique
- Feedback visual para cada tentativa
- Várias rodadas para desafiar os jogadores
- Design moderno e atraente

## Tecnologias Utilizadas
- React para a construção da interface do usuário
- CSS para estilização
- React Router para navegação entre páginas
- React Icons para ícones visuais
- React Hash Link para navegação suave dentro da página
- React Redux para gerenciamento de estado

## Como instalar e rodar o projeto
1. Clone o repositório;
2. Navegue até o diretório do projeto;
3. Instale as dependências usando `npm install`;
4. Inicie o servidor de desenvolvimento com `npm run dev`;
